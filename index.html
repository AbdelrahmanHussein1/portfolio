Automated ECG Arrhythmia Detection

Machine Learning • Healthcare AI

Project Overview

Developed a sophisticated neural network-based system for automated detection and classification of cardiac arrhythmias from ECG signals. This project addresses critical healthcare challenges by providing accurate, real-time cardiac anomaly detection using state-of-the-art deep learning techniques.

Key Features

  • Multi-class arrhythmia classification using deep neural networks
  • Real-time ECG signal processing and analysis
  • Advanced preprocessing with noise reduction and normalization
  • High accuracy detection with PyTorch Lightning framework
  • Hyperparameter optimization using Optuna
  • Cross-validation for robust performance evaluation

Technologies Used

Python PyTorch PyTorch Lightning Neural Networks Signal Processing Healthcare AI Optuna

Impact & Results

Successfully achieved high-accuracy classification of various cardiac arrhythmias, demonstrating the potential for AI-assisted medical diagnosis. The system can help healthcare professionals make faster, more accurate diagnoses and improve patient outcomes.

GenZ to Formal English Translation

Natural Language Processing • Deep Learning

Project Overview

Built an end-to-end NLP system for multi-directional style transfer between Gen Z slang, normal English, and formal/professional writing. This innovative project combines traditional sequence-to-sequence models with modern transformer architectures to achieve high-quality text translation.

Key Features

  • Multi-directional style transfer (GenZ ↔ Formal ↔ Normal)
  • Seq2Seq LSTM with Bahdanau Attention mechanism
  • Transformer T5 model implementation
  • Interactive Gradio UI with interactive slang highlighting
  • BLEU score evaluation for translation quality
  • Custom dataset creation and preprocessing

Technologies Used

Python NLP Transformers Seq2Seq LSTM Attention Mechanism Gradio T5

Impact & Results

Created a powerful tool for language style transfer that can help bridge communication gaps between different demographics. The system demonstrates high translation quality across multiple style directions and provides an intuitive user interface for real-world applications.

AeroScan – Aerial Image Segmentation

Computer Vision • Image Processing

Project Overview

Developed a comprehensive digital image processing system for aerial image analysis and land-cover segmentation. AeroScan combines classical computer vision techniques with clustering algorithms to perform accurate terrain classification from aerial photographs.

Key Features

  • Intelligent land-cover segmentation using K-means clustering
  • Fast Fourier Transform (FFT) analysis for frequency domain processing
  • Interactive MATLAB GUI for user-friendly operation
  • Multi-class terrain classification (vegetation, water, urban, etc.)
  • Advanced image preprocessing and enhancement
  • Real-time visualization of segmentation results

Technologies Used

MATLAB Computer Vision Image Processing K-means Clustering FFT Analysis GUI Development

Impact & Results

Successfully implemented a robust aerial image segmentation system with practical applications in urban planning, environmental monitoring, and agricultural analysis. The intuitive GUI makes advanced image processing accessible to non-technical users.

Smart Campus Navigation System

Algorithms • Graph Theory

Project Overview

Created an intelligent campus navigation system that helps students and visitors find optimal routes across campus. The application leverages graph algorithms to compute both shortest and fastest paths while providing an intuitive user interface.

Key Features

  • Shortest path calculation using Dijkstra's algorithm
  • Fastest route optimization considering walking speeds
  • Interactive campus map visualization
  • Multi-building routing with elevation considerations
  • Real-time distance and time estimates
  • User-friendly interface with location search

Technologies Used

Python Graph Algorithms Dijkstra's Algorithm Data Structures UI/UX Design Optimization

Impact & Results

Delivered a practical solution that significantly improves campus navigation efficiency. The system helps students save time and reduces confusion when navigating between buildings, especially beneficial for new students and visitors.
